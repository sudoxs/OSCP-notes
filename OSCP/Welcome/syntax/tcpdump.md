

📊 جدول فلگ‌های مهم tcpdump

| فلگ            | معنی                 |
| -------------- | -------------------- |
| `-i eth0`      | انتخاب اینترفیس      |
| `-n`           | بدون resolve DNS     |
| `-nn`          | بدون DNS و پورت      |
| `-v`           | verbose              |
| `-vv`          | خیلی verbose         |
| `-X`           | hex + ASCII          |
| `-A`           | ASCII فقط            |
| `-c 100`       | فقط 100 پکت          |
| `-w file.pcap` | ذخیره برای Wireshark |
| `-r file.pcap` | خواندن فایل          |

📊 فیلترهای کلاسیک tcpdump

| هدف      | فیلتر               |
| -------- | ------------------- |
| IP خاص   | `host 192.168.1.10` |
| فقط مبدا | `src host 10.0.0.5` |
| فقط مقصد | `dst host 8.8.8.8`  |
| پورت     | `port 80`           |
| TCP      | `tcp`               |
| UDP      | `udp`               |
| DNS      | `port 53`           |
| HTTP     | `port 80`           |
| HTTPS    | `port 443`          |
